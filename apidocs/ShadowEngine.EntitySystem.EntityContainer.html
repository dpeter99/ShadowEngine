Class EntityContainer

Entity container is a memory manager for a single type of Entity This creates EntityContainer::MemoryChunks that contain a block of memory for MAX_OBJECTS_IN_CHUNK number of entities, when it gets full it creates a new MemoryChunk. These are created by the EntityManager for each entity type that gets registered.
